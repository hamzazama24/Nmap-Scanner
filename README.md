Wireshark DNS Traffic Analysis
Overview

This project involved capturing and analyzing network traffic using Wireshark on a Wi-Fi interface. The primary focus was examining DNS traffic to understand how domain name resolution works and how different network protocols interact during communication.

Captured live network traffic using Wireshark
Filtered and analyzed DNS packets
Inspected protocol layers including:

IP

DNS
Examined source and destination IP addresses
Analyzed DNS queries and responses
Observed real-world web traffic behavior

Findings

During the analysis, DNS queries and responses were examined for domains such as zorvian.com and Cloudflare-hosted services. The captured traffic demonstrated normal domain resolution processes, providing insight into how devices translate domain names into IP addresses before establishing network connections.

Identifying protocol interactions across network layers
Interpreting packet-level data for troubleshooting and security analysis
